# Multi-layer Perceptron Experiment

## 编程环境

实验使用的编程环境如下：

- Python 3.8
- PyTorch 2.0.1
- Matplotlib 3.7.2
- NumPy 1.23.4

## 实验参数

在实验中，我们使用了以下参数和超参数：

- 最大学习率（learning_rate）：0.01
- 10 折交叉验证
- 随机种子（random_state）：42
- optimizer
- learning rate
- hidden_layer_sizes
- max_epochs
